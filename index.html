Spaceflight Mission Tracker

My Final Project for Harvard University's CS50x. A Flask Web application that integrates a 3rd party API to let you view, favourite and set alerts for upcoming Space Launches.

Python Flask SQLAlchemy
More info

Due to CS50's rules I am unable to share the code publicly for this project. However I can share the Final Project Demonstration which can be viewed via the youtube video below.

This is a Flask Web Application that fetches and displays upcoming SpaceX and other Spaceflight launches using the Launch Library 2 API. API responses are cached in a local SQLite database to reduce calls and stay within rate limits. Unauthenticated users can only view SpaceX launches on the homepage. Visitors can register for an account on the website. More spaceflight launch information is available to view for authenticated users. Once registered, users can log in and view mission details for all upcoming space missions from all agencies. Authenticated users can favourite missions to view later on their dashboard. Authenticated users can also set alerts for upcoming launches.
